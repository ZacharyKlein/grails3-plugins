About

grails-plugins.org is a static website for searching Grails 3 plugins. It pulls data from Bintray and Github APIs into a unified interface to make it quick and easy to find and use Grails plugins. All plugin data is updated hourly.

Follow @grails_plugins on Twitter to keep up with the latest plugin releases.

This website is open source.
